# HTML & CSS Practice 🚀

This repository contains my HTML and CSS practice projects created while learning front-end web development. It includes various webpages and UI components to strengthen my understanding of HTML structure, CSS styling, layouts, and responsive design.

## 🛠️ Technologies Used

- HTML5
- CSS3
- Visual Studio Code
- Git & GitHub

## 🎯 Topics Covered

- HTML Structure
- Semantic HTML
- Forms & Input Elements
- Tables
- CSS Selectors
- Box Model
- Flexbox
- Positioning
- Colors & Typography
- Cards & Layouts

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Tanisha-crypto/Daily_Practice.git
```

2. Open the project folder in Visual Studio Code.

3. Open any `.html` file in your browser.
